Map Application

Developed a fully interactive map application using Google Maps API and the browser’s Navigator API to provide a dynamic, location-aware user experience.

Key Features:
  • Displayed multiple markers using AdvancedMarkerElement, with location data from City of Hamilton public library JSON datasets.
  • Enabled user geolocation via the Navigator API with a custom marker showing current position.
  • Integrated Google Maps Directions API for turn-by-turn directions between selected locations.
  • Used Fetch API to retrieve data from public APIs and dynamically populate the map.
  • Implemented info windows displaying detailed location information for each marker.
Technical Highlights:
  • JavaScript for dynamic data fetching, DOM manipulation, and interactions
  • Google Maps API for rendering maps, markers, and directions
  • Browser Navigator API for geolocation
  • JSON data integration from public datasets
  • Responsive interface styled with Bootstrap
